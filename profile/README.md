# reverse-craft

AI-powered reverse engineering toolkit.

[中文](#reverse-craft-1)

## What is this?

A collection of tools and libraries designed for reverse engineering, with a core goal of making AI your reverse engineering partner.

We build tools that solve real problems AI faces in reverse engineering scenarios: code too long, messy structure, lack of context. Through intelligent processing and MCP protocol, we enable AI assistants to truly understand and analyze obfuscated code.

## Repositories

| Repository | Description |
|------------|-------------|
| [smart-fs](https://github.com/reverse-craft/smart-fs) | Core code processing library - beautify, truncate, search, analyze |
| [smart-fs-mcp](https://github.com/reverse-craft/smart-fs-mcp) | MCP server - AI tool wrapper for smart-fs |
| [ai-tools](https://github.com/reverse-craft/ai-tools) | AI analysis toolkit - LLM-driven code analysis capabilities |

More tools in development.

# reverse-craft

AI 辅助逆向工程工具套件。

## 这是什么？

一系列为逆向工程设计的工具和库，核心目标是让 AI 成为你的逆向搭档。

我们构建的工具解决 AI 在逆向场景下的实际问题：代码太长、结构混乱、缺少上下文。通过智能处理和 MCP 协议，让 AI 助手能够真正理解和分析混淆代码。

## 仓库

| 仓库 | 说明 |
|------|------|
| [smart-fs](https://github.com/reverse-craft/smart-fs) | 核心代码处理库 - 美化、截断、搜索、分析 |
| [smart-fs-mcp](https://github.com/reverse-craft/smart-fs-mcp) | MCP 服务器 - smart-fs 的 AI 工具封装 |
| [ai-tools](https://github.com/reverse-craft/ai-tools) | AI 分析工具集 - LLM 驱动的代码分析能力 |

更多工具持续开发中。
